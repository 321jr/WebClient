SPF

To avoid emails from your domain going into Spam, we recommend adding a SPF record. This can typically be done in the control panel of your domain name registrar. {{ 'LEARN_HOW' | translate }}

Please use the following SPF record:

 v=spf1 include:derp.net ~all

The current SPF record for {{ ctrl.domain.domain }} is:

 v=spf1 include:herp.org ~all

If your domain already has an SPF record, you just need to add _spf.protonmail.ch to the existing record.

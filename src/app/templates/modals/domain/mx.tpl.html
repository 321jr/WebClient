4. {{ 'MX_RECORD' | translate }}

Before you can receive emails for your custom domain addresses at ProtonMail, you need to add the following MX record to the DNS of {{ ctrl.domain.DomainName }}.

This can typically be done in the control panel of your domain name registrar. {{ 'LEARN_MORE' | translate }}

Add this MX record:

Host Priority Value
@ 10 * mail.protonmail.ch

Delete any other MX records or make sure ProtonMail's Priority is the lowest number.

If {{ ctrl.domain.DomainName }} is actively receiving emails, create all used email addresses in Step 3 before changing MX record to ensure a smooth transition.

Note, it could take up to an hour after updating your DNS record before verification succeeds.

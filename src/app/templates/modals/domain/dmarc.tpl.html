6. DMARC {{ 'NOT_SET' | translate }} {{ 'SET!' | translate }}

If you have set both SPF and DKIM, DMARC allows you to specify how other email services should deliver email for your domain if SPF or DKIM checks failed. This can make it harder for spammers pretending to be you but may also cause delivery issues if not done properly. {{ 'LEARN_MORE' | translate }}

Typical DMARC record:

Host Value
_dmarc v=DMARC1; p=none; rua=mailto:address@yourdomain.com

The current DMARC record for {{ ctrl.domain.domain }} is:

p=none is report only mode so no effect on email delivery

address@yourdomain.com is where you will receive DMARC reports from other email services

DMARC

For improved protection against spam, you can add the following DMARC record. This typically be done in the control panel of your domain name registrar. {{ 'LEARN_HOW' | translate }}

Please use the following DMARC record:

v=DMARC1; p=none; 
rua=mailto:dmarc_agg@auth.returnpath.net; 
ruf=mailto:dmarc_afrf@auth.returnpath.net; 
rf=afrf; pct=100
